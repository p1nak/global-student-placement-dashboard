# 🎓 Global Student Placement Analysis Dashboard

An interactive **data analysis and visualization dashboard** built using **Python, Dash, Plotly, and Pandas** to explore global student placement outcomes, salary trends, and career insights.

This project analyzes how **academic performance, internships, skills, and specialization influence placement opportunities and salary packages**.

---

# 📊 Dashboard Preview

![Dashboard Preview](images/dashboard.png)

The dashboard provides insights into:

- Placement rate by college tier
- Salary distribution
- CGPA vs salary relationship
- Country-wise placement trends
- Industry hiring patterns
- Feature correlations

## 📊 Visual Analytics

The dashboard includes interactive charts:

- Placement Rate by College Tier

- CGPA vs Salary

Users can filter the dashboard by:

# 🧠 Analytical Questions Explored

This project answers questions such as:
